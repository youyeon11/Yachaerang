# 🍅 야채랑 🥦

농수산물 가격 정보 제공과 농업 도우미 플랫포 야채랑을 소개합니다.

## 프로젝트 소개

이 플랫폼은 농업인과 농산물 관련 종사자들을 위한 종합 정보 서비스입니다. 실시간 농수산물 가격 정보, AI 챗봇을 통한 농업 상담, 농촌 관련 기사 제공 등 다양한 기능을 제공합니다.
