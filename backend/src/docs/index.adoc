== 개요

이 문서는 API의 사용 방법을 안내합니다.

[[overview-http-status-codes]]
=== HTTP 상태

|===
| 상태 코드 | 설명

| 200 OK
| 요청 성공

| 400 Bad Request
| 잘못된 요청

| 401 Unauthorized
| 인증 실패

| 404 Not Found
| 리소스를 찾을 수 없음

| 500 Internal Server Error
| 서버 오류
|===

[[overview-http-codes]]
=== HttpStatus 응답 필드의 코드

|===
| 상태 코드 | 설명

| OK
| 요청 성공

| NO_CONTENT
| 요청은 성공했으나 빈 본문

| NOT_FOUND
| 리소스를 찾을 수 없음

| INTERNAL_SERVER_ERROR
| 서버 오류
|===


[[overview-response-structure]]
=== 공통 응답 구조

모든 API 응답은 다음과 같은 공통 구조를 따릅니다:

[source,json]
----
{
  "httpStatus": "OK",
  "success": true,
  "code": "OK",
  "message": "요청에 성공하였습니다.",
  "data": { }
}
----

|===
| 필드 | 타입 | 설명

| httpStatus
| String
| HTTP 상태 코드 이름

| success
| Boolean
| 요청 성공 여부

| code
| String
| 애플리케이션 레벨 응답 코드

| message
| String
| 응답 메시지

| data
| Object
| 실제 응답 데이터 (성공 시)
|===
