HTTP 상태

상태 코드 설명

상태 코드 설명

공통 응답 구조

모든 API 응답은 다음과 같은 공통 구조를 따릅니다:

{
  "httpStatus": "OK",
  "success": true,
  "code": "OK",
  "message": "요청에 성공하였습니다.",
  "data": { }
}

로그인 요청 API

POST 요청을 통해 로그인을 요청합니다.

Curl request

$ curl 'http://localhost:8080/api/v1/auth/login' -i -X POST \
    -H 'Content-Type: application/json;charset=UTF-8' \
    -H 'Accept: application/json' \
    -d '{
  "email" : "test@test.com",
  "password" : "1234"
}'

HTTP request

POST /api/v1/auth/login HTTP/1.1
Content-Type: application/json;charset=UTF-8
Accept: application/json
Content-Length: 54
Host: localhost:8080

{
  "email" : "test@test.com",
  "password" : "1234"
}

HTTP response

HTTP/1.1 200 OK
Content-Type: application/json;charset=UTF-8
Content-Length: 220

{
  "httpStatus" : "OK",
  "success" : true,
  "code" : "200",
  "message" : "요청이 성공적으로 처리되었습니다.",
  "data" : {
    "accessToken" : "access-token",
    "refreshToken" : "refresh-token"
  }
}

HTTPie request

$ echo '{
  "email" : "test@test.com",
  "password" : "1234"
}' | http POST 'http://localhost:8080/api/v1/auth/login' \
    'Content-Type:application/json;charset=UTF-8' \
    'Accept:application/json'

즐겨찾기(관심 대시보드) 관련 API

즐겨찾기 등록 API

POST 요청을 통해 관심있는 대시보드에 대하여 등록해주세요. RequestBody와 Header를 입력해주세요.

Curl request

$ curl 'http://localhost:8080/api/v1/favorites' -i -X POST \
    -H 'Content-Type: application/json;charset=UTF-8' \
    -H 'Authorization: Bearer test-access-token' \
    -H 'Accept: application/json' \
    -d '{
  "productCode" : "PROD001",
  "periodType" : "DAILY"
}'

HTTP request

POST /api/v1/favorites HTTP/1.1
Content-Type: application/json;charset=UTF-8
Authorization: Bearer test-access-token
Accept: application/json
Content-Length: 57
Host: localhost:8080

{
  "productCode" : "PROD001",
  "periodType" : "DAILY"
}

HTTP response

HTTP/1.1 200 OK
Content-Type: application/json;charset=UTF-8
Content-Length: 227

{
  "httpStatus" : "OK",
  "success" : true,
  "code" : "200",
  "message" : "요청이 성공적으로 처리되었습니다.",
  "data" : {
    "favoriteId" : 1,
    "productCode" : "PROD001",
    "periodType" : "DAILY"
  }
}
